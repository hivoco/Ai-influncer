## Icon Usage

- Always use **Lucide React** icons instead of inline SVG elements
- Import icons from `lucide-react` package

## Code Style

- Use the least amount of JSX while maintaining a working UI
- Keep components clean and minimal
- Prioritize readability and maintainability
